# R Projects Portfolio

A collection of data science and machine learning projects built in R, spanning predictive modeling, clustering, optimization, and exploratory data analysis.

## Projects

### Machine Learning & Classification
| Project | Description |
|---|---|
| `Fetal_Health_Prediction_Model.Rmd` | Multi-class classification to predict fetal health outcomes from cardiotocography data |
| `knn_breast_cancer_classification.r` | K-Nearest Neighbors model for breast cancer diagnosis |
| `c50_decision_tree_heart_classification.R` | C5.0 decision tree for heart disease classification |
| `c50_random_forest_credit_default.R` | Random forest model for predicting credit default |
| `naive_bayes_credit_default_roc.R` | Naive Bayes classifier with ROC curve evaluation for credit default |

### Clustering & Unsupervised Learning
| Project | Description |
|---|---|
| `Financial_Behavior_Cluster_Analysis.Rmd` | K-Means clustering on financial behavior data |
| `kmeans_clustering_teen_social_media.r` | Segmenting teen social media usage patterns via K-Means |
| `kmeans_clustering_with_c50_imputation.R` | K-Means clustering with C5.0-based missing value imputation |

### Association Rules
| Project | Description |
|---|---|
| `apriori_association_rules_groceries.r` | Market basket analysis on grocery transaction data |
| `apriori_association_rules_heart_data.R` | Apriori association rules applied to heart health data |

### Regression & Statistical Analysis
| Project | Description |
|---|---|

| `S&P500_Financial_Performance_Analysis.Rmd` | Regression and financial performance analysis on S&P 500 data |

### Optimization
| Project | Description |
|---|---|

### Data Wrangling & EDA
| Project | Description |
|---|---|
| `Basketball_Player_Data_Profiling.RMD` | Exploratory data analysis and profiling of basketball player stats |
